Welcome to theastrologer-api
============================

`theastrologer-api <http://sandipbgt.com/theastrologer/api>`_ is an
un-official REST API that exposes readonly GET endpoints to see your horoscopes
from `theastrologer.com <http://theastrologer.com>`_ website

The api is written in `Flask <http://flask.pocoo.org>`_ framework using
`theastrologer <https://pypi.python.org/pypi/theastrologer>`_ python library. It
also allows you to send today's horoscope as SMS using `Twilio API <http://twilio.com>`_

The code is open source, `available on github <https://github.com/sandipbgt/theastrologer-api>`_.
